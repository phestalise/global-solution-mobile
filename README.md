# 🌾 AgroTech

Aplicativo mobile desenvolvido para monitoramento agrícola inteligente, integrando uma API em C# com banco de dados para fornecer informações estratégicas a produtores rurais.

---

# 📖 Sobre o Projeto

O **AgroTech** é uma solução mobile criada para auxiliar pequenos e médios produtores rurais no acompanhamento de suas propriedades através de dados agrícolas, climáticos e satelitais.

A aplicação permite visualizar indicadores importantes da lavoura, acompanhar propriedades cadastradas e receber alertas que auxiliam na tomada de decisões para aumentar a produtividade e reduzir riscos.

O sistema é composto por:

* Aplicativo Mobile desenvolvido em React Native.
* API REST desenvolvida em C# (.NET).
* Banco de Dados Oracle para armazenamento das informações.
* Integração entre front-end, back-end e banco de dados.

# 🚀 Funcionalidades

## 🔐 Autenticação

* Cadastro de produtores.
* Login utilizando CPF e senha.
* Persistência de sessão.

## 📊 Dashboard

* Monitoramento da situação da lavoura.
* Visualização de indicadores agrícolas.
* Exibição de alertas recentes.
* Acompanhamento das informações cadastradas.

## 🚜 Gestão de Propriedades

* Cadastro de propriedades rurais.
* Atualização de informações.
* Consulta de propriedades cadastradas.

## 🛰️ Monitoramento Agrícola

* Consulta de leituras agrícolas.
* Acompanhamento de indicadores da plantação.
* Histórico de informações registradas.

## ⚠️ Sistema de Alertas

* Alertas agrícolas.
* Alertas climáticos.
* Níveis de criticidade.

## 👤 Perfil do Produtor

* Consulta de dados pessoais.
* Atualização de informações cadastrais.

# ⚙️ Como Executar o Projeto

## 1. Clonar o repositório

```bash
git clone https://github.com/phestalise/global-solution-mobile.git
```

## 2. Acessar a pasta do projeto

```bash
cd AgroTech
```

## 3. Instalar as dependências

```bash
npm install
```

## 4. Iniciar o aplicativo

```bash
npm start
```

---

## 5. Executar no dispositivo

Após iniciar o Expo:

* Android Emulator: pressione `a`
* iOS Simulator: pressione `i`
* Dispositivo físico: escaneie o QR Code utilizando o Expo Go
